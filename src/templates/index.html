ChatBot PDF

Esse projeto permite que você faça upload de arquivos PDF, busque e interaja com o conteúdo dos documentos usando um modelo de chatbot IA. Depois disso, enviamos as conversas para um banco de dados local e damos uma opção para transferir as respostas da IA, para um arquivo excel e depois envia-lo para um email. Basta fazer o upload dos seus arquivos e fazer suas perguntas no chat.
